# TagMatch Pro (v1.9) — 遇见同频的人

> **“先懂自己，再遇同频。”**

TagMatch Pro 是一款基于深度标签构建的精神画像匹配工具。它告别传统的“看脸划卡”模式，通过 3D 标签云探索和白盒匹配算法，帮助用户在数字海洋中精准找到灵魂契合的伙伴。

## ✨ 核心特性

- **🌌 3D 标签宇宙**：直观的球体标签云，支持拖拽旋转，沉浸式探索兴趣领域。
- **🧬 白盒匹配算法**：拒绝黑箱推荐。我们公开匹配公式 `S = Σ(w₁·Tags) + Σ(w₂·Cluster) + Σ(w₃·Geo)`，让你清楚知道为什么被匹配。
- **🎨 现代极致设计**：采用 Glassmorphism（玻璃拟态）与沉浸式动画，支持深色模式。
- **🔒 隐私优先**：敏感数据本地化处理，加密传输。

## 🛠️ 技术实现

- **前端**: HTML5, JavaScript (ES6+), Tailwind CSS
- **可视化**: 3D Sphere Tag Cloud Algorithm (原生 Canvas 渲染)
- **动效**: CSS3 Keyframes, Scroll-smooth

## 🚀 快速开始

只需在浏览器中打开 `index.html` 即可运行体验。
